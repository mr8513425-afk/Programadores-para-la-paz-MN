repocitorio: espacio de almacenamiento en la nube donde se guarda el código fuente, archivos y el historial de cambios de un proyecto, permitiendo el control de versiones, trabajo colaborativo y gestión de proyectos.
commit:  El proceso clave es: usar git add para preparar los cambios y git commit -m "mensaje" para registrarlos, asegurando que el historial tenga descripciones claras
push: Para subir código a GitHub, utilizas el comando git push para cargar tus confirmaciones (commits) locales a un repositorio remoto
